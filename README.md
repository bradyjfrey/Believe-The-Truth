# Believe the Truth

A 1 vs All (asymmetrical) multiplayer survival-horror game on Roblox. For collaborative future work, the game is generated by Argon using VS code. Future project details will expand here as we build. 

## Getting Started

To build the place in the project root use:

```bash
argon build
```

To begin syncing open Roblox Studio and start Argon server using:

```bash
argon serve
```
