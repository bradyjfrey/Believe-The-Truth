# 🎨 Art & Sound Build List

Stuff the game needs from the art team. Build everything **in Workspace** like always.
**No scripts inside models.** The names below matter: when your effect gets moved in,
the game finds it *by its exact name*, so keep the name with your build.

**Tip for effects:** an "effect" is just a Part or Model with the fancy stuff inside it,
like ParticleEmitters, PointLights, and Sounds. Build it, name it, done. The game handles
placing it, playing it, and cleaning it up.
